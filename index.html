Find method

Find method returns the first elemet of the array as per the condition provided. It is similar to filter method.

But the difference is given as follow

  1. Filter method returns all the elements that match the condition while the find method only returns the first one
  2. Filter methodreturns new array while find method only returns the element itself(not array)
